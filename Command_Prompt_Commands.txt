COMMAND PROMPT COMMANDS
---------------------------
dir =====> #To see a directory
cd <filename> =====> #To go into a directory
cd .. =====> #To go back one directory
mkdir <folder name> =====> #To create a folder
echo. > <filename> =====> #To create a file
